This is a serie of classes to interact with the Iannix application : 


http://www.iannix.org/fr/index.php


It's mainly a wrapper around OSCresponders,  listening to Iannix application messages.


It's supposed to help you to get and manage osc messages  from Iannix, also to create objects from sclang onto the Iannix score, and to get control  on transport actions.


Note: I suggest to thoroughly read Iannix application documentation at first, as anything in there depends on Iannix application implementation.


Note also that Iannix is still in a developpement stage and so anything in those classes is subject to be  changed/improved, eventually be broken someday... if not already so :)


Please report  any bug and dumb things that might append (or appear to you in the code !), thanks.
